# 📖 About ZXFathan CyberSec Academy

<p align="center">
<img src="https://zxfathn.github.io/zko.png" width="750">
</p>

ZXFathan CyberSec Academy is a gamified cybersecurity learning application developed for Android devices. The platform is designed to make cybersecurity education more interactive, engaging, and accessible for students, beginners, and technology enthusiasts.

Users can learn cybersecurity concepts through structured learning modules, practical simulations, quizzes, and interactive challenges while earning XP, leveling up, unlocking achievements, collecting badges, and competing on leaderboards.

The goal of this project is to bridge the gap between theoretical learning and practical cybersecurity experience through an immersive mobile learning experience.
