## ⚡ About Project

ZXFathan Academy is a modern cybersecurity learning platform designed to help beginners learn cyber security through gamification. The platform combines learning materials, quizzes, practical simulations, challenges, rankings, badges, achievements, and progression systems into a single interactive experience.

Users can improve their cyber security skills while earning XP, leveling up, unlocking achievements, and completing real-world inspired challenges.
