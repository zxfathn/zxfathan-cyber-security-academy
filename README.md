## About ZXFathan Academy

<p align="center">
<img src="https://zxfathn.github.io/zko.png" width="800">
</p>

ZXFathan Academy is a gamified cybersecurity learning platform designed to make learning cyber security more engaging, interactive, and accessible. The platform combines educational content, quizzes, simulations, and practical challenges into a single learning ecosystem.

Users can gain experience points (XP), level up, earn achievements, unlock badges, and compete on leaderboards while learning essential cyber security concepts and techniques.
